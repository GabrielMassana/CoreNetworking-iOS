A static library project that simplifies NSURLSession.

# How to Install it

#### Podfile

```ruby
platform :ios, '8.0'
pod 'CoreNetworking', '~> 1.0'
```
#### Old school

Drag into your project the folder `/CoreNetworking-iOS`. That's all.
